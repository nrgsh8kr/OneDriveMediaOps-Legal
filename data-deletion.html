How to Delete Your Data

Option 1: Automatic Deletion (Recommended)

Simply uninstall the OneDrive MediaOps app from your device. This will automatically delete:

  • All cached file metadata
  • All scan results
  • All app preferences and settings
  • All locally stored authentication tokens

On Android: Settings → Apps → OneDrive MediaOps → Uninstall

Option 2: Clear Data (Keep App Installed)

To delete local data without uninstalling:

  1. Open OneDrive MediaOps app
  2. Go to Settings (gear icon)
  3. Tap "Clear Cache" to remove file metadata
  4. Tap "Clear Scan History" to remove duplicate detection results
  5. Tap "Sign Out" to remove authentication tokens

Or use Android system settings:

Settings → Apps → OneDrive MediaOps → Storage → Clear Data

Types of Data Deleted

✓ Data That Will Be Deleted:

  • All local cache data: File names, sizes, modification dates, checksums
  • Scan history: Duplicate detection results and settings
  • App preferences: Theme, notification settings, algorithm choices
  • Authentication state: Login session and OAuth tokens

✗ Data We Cannot Delete (We Don't Have It):

  • Your OneDrive files: Stored by Microsoft, not by our app
  • Microsoft account data: Managed by Microsoft's privacy policies
  • File contents: Never transmitted to or stored by our app

Retention Period
